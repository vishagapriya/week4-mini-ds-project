# Week 4 Mini End-to-End Data Science Project

## Project Title
Mini End-to-End Machine Learning Pipeline

---

## Objective
To build a complete data science workflow including data preprocessing, exploratory data analysis (EDA), model building, evaluation, and insight generation.

---

##  Dataset
- Raw dataset was cleaned and preprocessed
- Final cleaned dataset used for training the model

---

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn
- Jupyter Notebook

---

## Workflow

### 1. Data Preprocessing
- Handled missing values
- Removed duplicates
- Encoded categorical variables

### 2. Exploratory Data Analysis (EDA)
- Data visualization using graphs
- Correlation analysis
- Pattern identification

### 3. Model Building
- Train-test split performed
- Machine learning model trained
- Model saved using joblib/pickle

### 4. Evaluation
- Evaluated using accuracy / F1-score
- Compared predictions with actual values

---

## Results
- Model performed well on test data
- Key features identified influencing predictions
